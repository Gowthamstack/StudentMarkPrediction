# Student Mark Prediction using Machine Learning

This project predicts student marks based on various academic and personal factors such as study hours, attendance, previous scores, assignments, and other relevant inputs using Machine Learning techniques.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature selection
- Model training and testing
- Marks prediction
- Performance evaluation using metrics

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook / VS Code

## Machine Learning Algorithms
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

## Goal
To help understand how different factors affect student performance and predict marks accurately using data-driven models.

## Future Improvements
- Web deployment using Flask / Streamlit
- Real-time prediction UI
- Better dataset with more features
